# AIDeazz - AI Agents Web3 Showroom

**Live Website:** https://aideazz.com

Elena Revicheva's portfolio showcasing AI agent projects and Web3 innovations.

## ?? Features

- Interactive portfolio showcasing AI agent projects
- Multi-language support (English/Spanish)
- Responsive design with modern UI/UX
- Integration with Web3 technologies
- Live demos of AI agents (EspaLuz, ALGOM, etc.)

## ??? Local Development

```sh
# Clone the repository
git clone https://github.com/ElenaRevicheva/aideazz.git

# Navigate to project directory
cd aideazz

# Install dependencies
npm install

# Start development server
npm run dev
```

## ?? Technologies

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, shadcn-ui
- **Deployment:** Fleek (IPFS)
- **i18n:** Multi-language support

## ?? Deployment

This site is automatically deployed to IPFS via Fleek from the `main` branch.

**Live URL:** https://aideazz.com
